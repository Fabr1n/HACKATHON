## 🎯 COMO USAR ESTA ESTRUTURA

### 1️⃣ DESENVOLVIMENTO

```
Para adicionar nova funcionalidade:

1. Criar rota no → backend/backendexpandido.js
2. Criar página no → frontend/nova-page.html
3. Adicionar dados no → dados/*.json
4. Documentar em → documentacao/TECNICO.md
5. Testar em → localhost:3000
```

### 2️⃣ ORGANIZAÇÃO

```
Cada pasta tem responsabilidade clara:

backend/        → Lógica, APIs, Algoritmos
frontend/       → UI, Formulários, Views
dados/          → Persistência, Base de dados
documentacao/   → Guias, Tutoriais, Referência
configuracao/   → Setup, Dependências, Secrets
```

### 3️⃣ MANUTENÇÃO

```
Pastas pequenas = Fácil manutenção
Separação clara = Fácil encontrar código
Documentação centralizada = Fácil entender
Dados separados = Fácil migrar banco
```
